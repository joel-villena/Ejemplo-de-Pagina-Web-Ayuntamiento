# Web de sostenibilidad de Granollers

Proyecto web desarrollado en **HTML y CSS** sobre sostenibilidad, medio ambiente y Agenda 2030, centrado en el municipio de **Granollers**. La página simula un portal informativo del ayuntamiento con diferentes apartados sobre medidas medioambientales, consejos, contacto ciudadano, 
galería de imágenes y relación con la Unión Europea y los ODS.

# Descripción

Este sitio web presenta información relacionada con la sostenibilidad local mediante una estructura sencilla, visual y fácil de navegar. Todas las páginas comparten un diseño común con encabezado, menú de navegación, contenido principal y pie de página.

El objetivo del proyecto es ofrecer una web informativa sobre:

- Noticias y eventos medioambientales
- Medidas sostenibles aplicadas en Granollers
- Consejos para ciudadanos y empresas
- Agenda 2030 y Objetivos de Desarrollo Sostenible
- Formulario de contacto y denuncias ambientales
- Galería de imágenes relacionadas con el entorno y la sostenibilidad

# Tecnologías utilizadas

- **HTML / XHTML 1.0 Strict**
- **CSS3**
- **Google Maps Embed**

> No se ha utilizado JavaScript ni frameworks externos.
